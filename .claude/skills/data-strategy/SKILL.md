# 한국 의약품 데이터 API 조사 방법론

## 평가 항목 (각 API별)

| 항목 | 확인 방법 |
|------|---------|
| API 존재 여부 | data.go.kr 검색 + 기관 공식 개발자 포털 |
| 인증 방식 | API 키 신청 / 협약 기반 / 무료 공개 |
| 일일 호출 한도 | 공식 문서 또는 활용 가이드 |
| 응답 포맷 | JSON / XML |
| 업데이트 주기 | 실시간 / 월별 / 연간 |
| 상업 이용 가능 여부 | 이용 약관 원문 확인 |
| 약품 커버리지 | 처방약만 / OTC 포함 / 건기식 포함 |

## 핵심 API 목록 (확인 대상)

### 식품의약품안전처 (mfds.go.kr)
- 의약품 허가정보 서비스
- 의약품 낱알식별 서비스
- 의약품 주의사항 (임부, 노인, 소아)
- 공공데이터포털 키워드: "식약처 의약품"

### 건강보험심사평가원 (hira.or.kr)
- DUR 점검 서비스 (병용금기·연령금기·임부금기·효능군중복)
- 요양기관 정보 서비스
- 특이사항: DUR API는 공공데이터포털 외 별도 협약 필요 가능성 높음

### 식품안전나라 (foodsafetykorea.go.kr)
- 건강기능식품 원료 정보
- 건강기능식품 제품 정보
- 공공데이터포털 키워드: "건강기능식품"

### 약학정보원 e약은요
- 약품 낱알 식별, 효능, 주의사항
- 공공데이터포털 또는 별도 계약 확인

## 해외 데이터 평가 기준

**DrugBank:**
- 상업 이용: 유료 라이선스 필요 (Academic vs Commercial 구분)
- 한국 약품 커버리지: 글로벌 약품 중심, 한국 제네릭은 일부만 포함
- 장점: 성분 기반 상호작용 데이터 풍부

**RxNorm (NIH):**
- 무료 공개 API
- 한국 약품: 미국 FDA 승인 약품 중심 → 한국 고유 제네릭 커버리지 낮음
- 활용: 성분명(INN) 기반 매핑 보조

**OpenFDA:**
- 무료 공개 API
- 약물 이상반응(AER) 데이터 풍부
- 한국 약품 직접 조회는 어려움, 성분명 기반 활용 가능

**Natural Medicines (TRC):**
- 건기식-약 상호작용 데이터베이스
- 상업 이용: 구독 기반 (연간 수천 달러 수준)
- MVP 단계 적합성: 비용 부담 높음 → V2 검토 권장

## 출력 형식

각 API를 표 형태로 정리 후 MVP 권장 스택을 명시.
커버 불가 영역(건기식-약 상호작용 등)을 별도 섹션에서 솔직하게 기술.

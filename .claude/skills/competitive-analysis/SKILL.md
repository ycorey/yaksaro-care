# 경쟁 분석 방법론

## 기능 비교 매트릭스

각 서비스를 다음 기능 항목으로 평가 (O/△/X):

| 기능 | 평가 기준 |
|------|---------|
| 처방약 관리 | 처방약 등록·조회·이력 관리 가능 여부 |
| OTC 관리 | 일반의약품 직접 등록 가능 여부 |
| 건기식 관리 | 건강기능식품 등록·관리 가능 여부 |
| 약-약 상호작용 | 복용약 간 상호작용 경고 기능 유무 |
| 건기식-약 상호작용 | 건기식과 약물 간 상호작용 분석 유무 |
| 복약 알림 | 복용 시간 알림 기능 유무 |
| 약사 연결 | 약사 또는 약국과 연동 기능 유무 |
| OCR 처방전 | 처방전 이미지 자동 인식 유무 |
| 데이터 자동 수집 | 처방 이력 자동 연동(건강보험 등) 유무 |
| 앱 품질 (UX) | 앱스토어 평점 + 디자인 수준 |

## 앱 리뷰 페인포인트 추출

구글플레이·앱스토어에서 각 서비스의 1-2성 리뷰를 조사:
- 공통 불만: "OTC를 등록할 수 없다", "알림이 부정확하다", "인터페이스가 복잡하다"
- 이 페인포인트가 약사로 케어의 차별화 기회가 된다

## 보완재 vs 경쟁재 판단 기준

**보완재**: 다른 목적을 위한 서비스 (닥터나우: 비대면 진료) → 처방 후 복약관리로 연결 가능
**경쟁재**: 동일 목적을 겨냥한 서비스 (심평원 앱) → 직접 전환 유도 필요

## 차별화 포인트 도출

다음 질문에 답변:
1. 기존 서비스 중 처방약+OTC+건기식을 통합 관리하는 곳이 있는가?
2. 약-약 상호작용 경고를 제공하는 무료 서비스가 있는가?
3. 약국이 환자에게 직접 추천하는 B2B2C 구조를 가진 서비스가 있는가?

## 출력 구조

1. 기능 비교표 (전체)
2. 서비스별 페인포인트 요약
3. 차별화 격차(Gap) 분석
4. 포지셔닝 권고
